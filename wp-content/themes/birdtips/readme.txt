== Description ==
BirdTIPS is a flexible three-column blog theme. The theme uses CSS3 media queries for its responsive layout and is therefore optimized for tablet pcs and smart phones. Features include the page navigation and breadcrumb navigation. You can also choose the link color, article title color and navigation menu color.

=== Features ===
Responsive Layout, Theme Settings, Page Navigation, Breadcrumb Navigation


=== Widgets Areas ===
The Theme has 2 customizable sidebars.
You can use these area to customize the content of your website.
If no content is be added to the sidebars these will display search box and category list, monthly archives, meta.
